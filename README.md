# Character-Continuity-Manager

Automatically track and preserve character continuity in SillyTavern.

CCM uses AI-assisted state extraction to maintain accurate character
details such as appearance, clothing, location, mood, relationships,
and scene status. Character data persists across sessions, supports
manual editing, archiving, re-import detection, and multi-character
cards.

✓ AI-powered character state tracking
✓ Appearance and clothing tracking
✓ Location and scene continuity
✓ Relationship tracking
✓ Character database
✓ Archive and restore system
✓ Re-import detection
✓ Multi-character card support
✓ OpenAI-compatible extraction models
✓ Current model extraction mode
✓ Manual editing and field locking
✓ History and snapshots
✓ Import and export support
